# Deep Learning for Defect Detection

This project focuses on building a deep learning model to detect defects in industrial X-ray images.

### Project Details:
- **Technologies Used**: Python, TensorFlow, OpenCV, AWS
- **Methodology**: Autoencoders and CNNs for defect detection
- **Challenges Overcome**: Achieved real-time defect detection with high accuracy.

### Results:
- **Accuracy**: 95%
- **Deployment**: The model was deployed on AWS using Docker and Kubernetes for scalability.
